Ng’Iworui a Esukul School clothes Les vêtements d'école

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

Dennis Lokitare

Turkana

Level 1

The audio for this story is currently not available.

Ewoi eworu lo.

This dress is long.

Cette robe est longue.

Epolot eworu lo.

This jersey is big.

Ce pull est grand.

Epolot abusek na.

This bag is big.

Ce sac est grand.

Aruba na.

This belt is…

Cette ceinture est…

Edit apukot na.

This hat is small.

Ce chapeau est petit.

Euriaka ng’asogis nu.

The socks are short.

Les chaussettes sont courtes.

Na kaneni ng’iketeat ng’amuk nu.

But these shoes are new.

Mais ces souliers sont neufs.

Itemokino jik.

… they fit well.

… ils font bien.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Dennis Lokitare
Language: Turkana
Level: Level 1
